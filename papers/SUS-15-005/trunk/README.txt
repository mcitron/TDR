Note that there is a file called .obsolete, which contains previous
versions of plots, tables, etc. Should be probably be deleted... 

"git tag" lists the tags
"git show <name>" gives the details of a tag
"git tag -a <name> -m 'comment'" creates an annotated tag 

"git tag -d <name>" deletes local tag
"git push <repo> :refs/tags/<name>" removes tag from remote repo
